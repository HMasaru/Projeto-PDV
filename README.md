# Projeto PDV (Ponto de Venda)

> 🎓 **Projeto Acadêmico / Extensionista**
> Repositório destinado aos projetos e atividades desenvolvidos para o curso de **Ciência da Computação da PUC**.

Sistema de **Ponto de Venda (PDV)** desenvolvido em **Python**. 

Este projeto tem como objetivo simular as operações rotineiras de um caixa comercial, gerenciando o registro de produtos, cálculo de totais e finalização de vendas. A estrutura do código foi pensada para ser modular e de fácil compreensão, aplicando conceitos sólidos de programação orientada a objetos e estruturação de software.

## 📂 Estrutura do Projeto

Os arquivos principais estão localizados dentro do diretório `PDV_Extensionista-master`. A aplicação foi dividida visando separar a lógica de negócios da interface (menus do terminal ou interface gráfica, dependendo da implementação).

## 🚀 Como Executar

Por ser um projeto em Python, não é necessário um processo de compilação prévia, apenas o interpretador da linguagem instalado na sua máquina (Python 3.x recomendado).

Abra o terminal, navegue até a pasta do projeto e execute o arquivo principal:

```bash
# 1. Entre no diretório do projeto
cd PDV_Extensionista-master

# 2. Execute o arquivo principal (Linux ou macOS)
python3 nome_do_arquivo_principal.py

# 2. Execute o arquivo principal (Windows)
python nome_do_arquivo_principal.py
